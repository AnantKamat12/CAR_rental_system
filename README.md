# Car Rental System (C++ | Object-Oriented Programming)

A simple yet well-structured **Object-Oriented Car Rental System** implemented in C++.  
This project demonstrates strong understanding of **classes, encapsulation, composition, dynamic object associations**, and state management.

---

## ✅ Features

- Add and manage cars in the system  
- Create customer profiles dynamically  
- Rent and return cars with availability tracking  
- Price calculation based on rental duration  
- User-friendly, menu-driven console interface  
- Demonstrates core OOP principles

---

## ✅ OOP Concepts Used

- **Classes & Objects** – Car, Customer, Rental, CarRentalSystem  
- **Encapsulation** – private members + public getters  
- **Constructors** – used to initialize objects cleanly  
- **Composition** – Rentals reference Car and Customer  
- **Abstraction** – clean APIs for renting and returning cars
